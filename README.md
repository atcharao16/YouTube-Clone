# YouTube Clone
- This is a *YouTube Clone* built using *HTML*, *CSS* and basic *JavaScript*.  
- The project focuses on replicating the user interface and layout of YouTube, including a responsive homepage and video page design.

# Features
- Responsive homepage layout
- Video thumbnails with title, channel name, and views
- Navigation bar similar to YouTube
- Clean and modern UI design
- Video page layout

# Technologies Used
- HTML5
- CSS3
- JavaScript

# What I learn 
- Building real-world UI using HTML & CSS
- Responsive layout using Flexbox
- Basic DOM event handling
